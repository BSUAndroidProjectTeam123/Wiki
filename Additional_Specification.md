# Дополнительная спецификация

* **Ограничения:** Приложение требует Android 8.0 (API level 26) или выше. Для получения колоды карт требуется стабильное интернет-соединение.
* **Безопасность:** Данные о результатах игр хранятся в изолированной песочнице приложения с помощью Room (SQLite). Ключи доступа к API не должны храниться в открытом виде.
* **Надежность:** При отсутствии интернета пользователь должен видеть уведомление об ошибке, а приложение не должно крашиться (обработка Exception).
